# Raw data and redistribution notice

This repository does not redistribute raw public sequencing data, raw single-cell/single-nucleus matrices, large Seurat/RDS objects, protected clinical records, or copyrighted article PDFs.

Raw datasets should be obtained from the original public repositories under the accession numbers reported in the manuscript. The repository contains locked gene sets, processed result tables, figure source data, analysis scripts, and environment files sufficient to inspect the analyses and regenerate manuscript-level summaries where the required public inputs are available.
